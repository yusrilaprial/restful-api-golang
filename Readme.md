# Learn Rest API with Go and Gin

This is a simple project to learn how to create a REST API with Go and Gin.

## How to Run

1. Clone the repository
2. Copy the `.env.example` file to `.env` and fill in the values
3. Create a vendor folder: `go mod vendor`
4. Run or build:
    - Run: `go run -mod=vendor main.go`
    - Build: `go build -mod=vendor`
